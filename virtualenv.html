Howto: virtualenv

Virtualenv lets us create fresh python installs that are kept completely separate from the rest of your machine, which is ideal for projects like Gryphon that have super-specific dependencies.

This sequence of commands installs virtualenv, initializes a fresh environment in the env/ directory, and turns it on.

$ pip install virtualenv
$ mkdir env && virtualenv -v env
$ source env/bin/activate

This turns the virtualenv off when you're finished:

$ deactivate
