Gryphon is launching soon, and I could use your help to find any last-minute bugs in the distribution or install process. If you have five minutes, it shouldn't take any more than that to install the framework and check if the unit tests are passing. Here's what to do:

Start a new python 2.7 virtualenv and install gryphon:

$ pip install gryphon-framework

And run the unit-test suite:

$ gryphon-runtests

You'll see a lot of cython build output, and then a summary of the test run. If any tests failed, copy the output and send it to me (email or messenger), along with with your operating system version and the output of $ pip freeze.

That's it, thanks a ton for helping out!
